# Movie App

A fully-featured cross-platform mobile app for discovering movies, searching titles, and managing a personal watchlist — built with React Native and Expo.

## 📱 Download
- [Android APK](https://expo.dev/artifacts/eas/5yCjjv1AUmnwBYwkT7UwEW.apk)
- [iOS (Expo Go)](https://expo.dev/accounts/abdulrahman-mohamd/projects/movie-app/updates/2f6f3eda-f466-485e-8983-b6d6943779f6)

## ✨ Features
- Browse and discover popular movies
- Search movies with a dynamic trending algorithm based on user searches
- Full movie details: poster, rating, description, release date, box office info
- Personal watchlist saved to database across sessions
- User authentication with profile management and custom avatar upload

## 🛠️ Tech Stack
- React Native, TypeScript, Expo SDK 54
- Expo Router (file-based navigation)
- NativeWind (Tailwind CSS for React Native)
- Appwrite (Auth, Database, Storage)
- TMDB API (movie data)

## ⚙️ Getting Started
```bash
npm install
npx expo start
```
